ce repo représente le contenu du dossier "src" de l'application
créée à l'aide de la commande : npx create-react-app my-app

aprés la création du create-react-app modifier le contenu 
du "src" par ces fichiers

le concept du jeu:
==> en cliquant sur "Jouer" le dé se lance et un nombre est apparu par hasard
==> si la face du dé représente le même nombre que le contenu du champ "face" qui est lui même choisi par hasard 
le jeu est fini et vous avez gagné sinon vous pouvez cliqué une autre fois sur "jouer" pour recommancer l'essai
==> le champ "nombre d'essai" calcul le nombre d'essais que vous avez effectué avant d'être gagner
